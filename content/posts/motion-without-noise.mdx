## Use motion to explain state

Hover glow, progress tracking, and subtle panel lift are enough to keep the UI feeling alive.

The moment motion becomes constant ambience, it starts competing with the text.

## Reduce amplitude, not personality

Short travel distances and soft opacity changes keep the interface crisp.

Restraint is what lets a cyber visual system survive everyday use.

## Design for reduced motion from the start

Every decorative animation should have a low-motion equivalent that preserves hierarchy without visual fatigue.

That constraint improves the system even for users who never enable accessibility settings.
